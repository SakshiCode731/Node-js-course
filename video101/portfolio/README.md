# My Portfolio Website

A professional and animated portfolio website built with HTML and CSS, featuring a dark theme.

## Features

- Dark theme background with animated twinkling stars
- Responsive design
- Advanced animations and transitions including:
  - Typing effect with gradient text for the main heading
  - Glowing floating photo animation
  - 3D hover effects on project cards with shimmer
  - Bouncing social links
  - Pulsing buttons
  - Staggered 3D slide-up for project cards
  - Animated background gradient
  - Enhanced hover effects on navigation and cards
- Fixed navigation header
- About section with photo and social links (Gmail, LinkedIn, GitHub)
- Projects showcase
- Contact form

## Setup

1. Place your photo in the `images/` folder and name it `your-photo.jpg`
2. Add project images in the `images/` folder (e.g., `project1.jpg`, `project2.jpg`, etc.)
3. Update the content in `index.html` to reflect your information:
   - Replace [Your Name] with your actual name
   - Replace [Your Profession], [Your Experience], [Your Completed Projects], and all placeholder text with your own details
   - Update the bio text and journey/experience entries
   - Update social links: replace `your-email@gmail.com`, `https://linkedin.com/in/your-profile`, `https://github.com/your-username` with your actual profiles
4. Customize colors and styles in `style.css` if needed

## Viewing the Website

Open `index.html` in your web browser to view the portfolio.

## Customization

- Edit `index.html` to change content
- Modify `style.css` for styling changes
- Add more projects by copying the project-card div in the projects section
